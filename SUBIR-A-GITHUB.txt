CIELO POSTRES - PASOS PARA SUBIR

1. Descomprime este ZIP.
2. Reemplaza TODOS los archivos del repositorio cielo-pene con estos archivos.
3. Sube también estos archivos importantes:
   - package-lock.json
   - .npmrc
   - vercel.json
   - .gitignore
4. Haz commit y push a la rama main.
5. En Vercel abre Deployments y ejecuta Redeploy without cache.

VARIABLES NECESARIAS EN VERCEL:
- MONGO_URI
- CLOUDINARY_CLOUD_NAME
- CLOUDINARY_API_KEY
- CLOUDINARY_API_SECRET
- GOOGLE_CLIENT_ID
- CLIENT_AUTH_SECRET

NO subas a GitHub:
- client_secret de Google
- archivos .env o .env.local
- contraseñas o claves privadas

RUTAS:
- Tienda: /
- Administrador: /admin
- Cuenta del cliente: /cuenta
